# 3D_Portfolio
I created my own portfolio using Three JS for 3D web development.  
In this React app I'm using react-three, emailjs for allowing e-mailing me, react-router-dom v6, react-tilt, framer-motion.  
I'm using Tailwind CSS for the styiling.  
The app is hosted here: https://aleksandromilenkov.netlify.app/
